Food Ordering Website

Created an online food ordering system using HTML and CSS that enables users to browse menus, select items, and place orders. Developed a user interface with responsive design, enhancing overall user experience. Implemented layout and styling techniques to ensure accessibility and visual appeal across various devices. A simple and responsive food ordering website where users can browse food items, add them to their cart, and place orders (without real payment processing). It is built using HTML, CSS, and JavaScript to provide an interactive user experience.
Key Features:
✅ Home Page – Displays featured restaurants or food categories.
✅ Menu Page – Users can browse available food items with images and prices.
✅ Cart System – Users can add or remove items from the cart dynamically using JavaScript.
✅ Order Summary – Shows total price and selected items before order confirmation.
✅ Responsive Design – Works on mobile, tablet, and desktop screens.
✅ Basic Animations – Smooth UI with CSS animations and JavaScript effects.
Tech Stack:
HTML → Structure of the website
CSS → Styling and layout (using Flexbox/Grid)
JavaScript → Interactive features (cart management, dynamic updates)
